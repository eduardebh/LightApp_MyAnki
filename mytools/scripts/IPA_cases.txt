# IPA_cases.txt
#
# Archivo para ir agregando casos de IPA/liaison (FR u otros idiomas).
# Lo consume el script: scripts/run_ipa_cases.py
#
# Formato por línea (separado por |):
#   texto | idioma | nota_opcional
# - texto: palabra o frase (puede tener espacios)
# - idioma: código ISO (fr, de, etc.). Default: fr
# - nota_opcional: cualquier comentario (no afecta ejecución)
#
# Ejemplos basados en los casos que has reportado:
elles viennent | fr | NO debería inventar liaison antes de consonante
ils sont allés | fr | evitar s‿ entre "ils" y "sont"
elles sont venues | fr | evitar s‿ entre "elles" y "sont"
